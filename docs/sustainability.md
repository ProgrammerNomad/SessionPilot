# Sustainability Model

## Licensing

SessionPilot is **fully open source, forever.**

- No "Pro" vs "Free" split
- No feature paywalls
- No license keys
- No activation servers
- All core features available to everyone

---

## Why Not a Freemium Model?

The freemium model creates:

- Incentive to cripple the free version
- Cloud dependency to enforce licensing
- Trust issues with the community
- Conflict with the self-hosted philosophy

SessionPilot's value comes from being **the trusted, fully-open alternative**. That trust cannot be bought back once lost.

---

## Possible Future Revenue Streams

These are optional, community-aligned, and never gate core functionality.

| Stream | Description |
|---|---|
| **Donations** | One-time donations via OpenCollective, Buy Me a Coffee, etc. |
| **GitHub Sponsors** | Monthly sponsorships from individuals and companies |
| **Support Plans** | Paid priority support for businesses (email/ticket) |
| **Managed Hosting** | Optional hosted version for users who want managed setup (separate product) |
| **Consulting** | Custom development or integration work |
| **Enterprise Support** | SLA-backed support contracts for organizations |

None of these require modifying the open-source plugin itself.

## Long-Term Goal

SessionPilot should be:

> **The go-to self-hosted session intelligence plugin for WordPress** -recommended by developers, trusted by privacy-conscious site owners, and maintained by a healthy open-source community.

Not a startup. Not a SaaS. A reliable open-source tool that outlasts any single vendor.
